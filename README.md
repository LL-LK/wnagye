# 桂林1944·命运应答文创产品数字化展示项目

## 项目概述

本项目是一个基于UniApp X的H5网页，用于"桂林1944·命运应答"文创产品的数字化展示。该网页作为比赛展示和汇报使用，非电商售卖或复杂系统，重点在于视觉统一性、文化氛围感及产品设计逻辑的清晰呈现。

## 技术栈

- **前端框架**：Vue 3
- **开发语言**：TypeScript
- **构建工具**：UniApp
- **样式方案**：CSS3

## 如何运行项目

### 前提条件

- 安装HBuilder X（标准版本）
- 确保电脑有网络连接

### 运行步骤

1. **导入项目**
   - 打开HBuilder X
   - 点击"文件" -> "导入" -> "从本地目录导入项目"
   - 选择项目所在目录：`f:\红旅\uni\红旅开发`
   - 点击"导入"按钮完成项目导入

2. **运行项目**
   - 在HBuilder X中，点击顶部工具栏的"运行"按钮
   - 选择"运行到浏览器" -> 选择你喜欢的浏览器（如Chrome、Firefox等）
   - HBuilder X会自动启动浏览器并加载项目

3. **查看项目**
   - 浏览器会自动打开项目页面，显示"桂林1944·命运应答"文创产品数字化展示
   - 你可以浏览各个区域，查看产品分类和详情

## 项目特点

1. **视觉统一性**：采用统一的设计风格和色彩方案，确保页面整体协调一致
2. **文化氛围感**：融入民国旧报刊元素、桂林山水意境和抗战文化记忆，营造浓厚的历史文化氛围
3. **交互体验**：流畅的动画效果和响应式设计，提供良好的用户体验
4. **性能优化**：图片懒加载和代码优化，确保页面加载迅速
5. **代码质量**：详细的代码注释和优化的代码结构，提高代码的可维护性
6. **兼容性**：支持主流浏览器，确保在不同设备上都能正常显示

## 主要功能

1. **首页头图Banner区域**：展示项目主标题和简介，使用指定背景图片
2. **项目理念区域**：阐述项目设计理念，融入历史相关视觉符号
3. **产品分类展示区域**：按照五大策略（呐喊者、燃灯者、织网者、渡河者、潜火者）展示产品，支持点击展开/收起
4. **单个产品详情展示区域**：展示产品高清大图、名称、设计寓意、历史意象、材质说明、尺寸规格、产品特点
5. **系列主题整合区域**：以时间轴形式展示五大策略主题
6. **结尾展示区域**：包含总结文本和项目相关单位信息

## 响应式设计
